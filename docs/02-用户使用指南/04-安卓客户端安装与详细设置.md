# 04. 安卓客户端安装与详细设置

TouchDX 的安卓客户端 (`MaiNative`) 不仅仅是一个简单的触摸屏面板，它还内置了众多细致的调节选项，以适配不同尺寸的平板或手机。

## 下载与安装
1.  前往 [YubaiNya/TouchDX-AndroidClient Releases](https://github.com/YubaiNya/TouchDX-AndroidClient/releases) 页面。
2.  下载最新的 `app_aligned.apk`。
3.  将其传到您的安卓设备并安装。如果遇到系统提示风险，请选择“继续安装”（因为这只是一个开发者自签名的开源软件）。

## 连接服务器
1. 打开应用，首屏会弹出一个连接设置对话框。
2. 如果您听从建议使用了 ADB 有线转发（即在电脑上执行了 `adb reverse`），请在输入框中保持默认的 **`127.0.0.1`**。
3. 点击“连接并启动悬浮窗”。

## 悬浮控制面板功能详解

连接成功后，屏幕左上角会出现一个半透明的 `M` 悬浮按钮。点击它可以展开强大的设置面板：

1.  **交互/游玩模式状态灯**：最上方会显示当前所处的模式（绿色表示交互，蓝色表示游玩，黄色表示正在连接诊断工具）。该状态由 PC 游戏端自动下发，无需人工干预。
2.  **锁定触摸 (穿透到桌面)**：非常实用的功能。开启后，不仅隐藏了 TouchDX 的面板，而且会让触摸事件**直接穿透**到下层的 Android 系统桌面。方便你在不退出 App 的情况下回复微信或切换歌曲。
3.  **校准 Touch**：开启校准模式。此时背景变暗，按键区域高亮。
    *   **拖拽**：单指按住屏幕可以平移整个打歌面板。
    *   **双指缩放**：可以放大或缩小打歌面板，以适应你平板的物理尺寸。
    *   修改满意后，再次点击“完成校准”即可保存。
4.  **重置布局位置**：如果不小心将面板拖到了屏幕外找不回来，点击此按钮可以将其瞬间居中恢复。
5.  **触摸半径 (Radius)**：调节屏幕判定圆的点大小。数值越大，你的手指越容易触发相邻的多个区块；数值越小，点击越精确。
6.  **画面缩放 (Scale)**：除了双指缩放，也可以通过这里的拉杆或 +/- 按钮进行极度微小的缩放微调。
7.  **游玩透明度 (Alpha)**：调节打歌面板的背景和线条透明度。如果你喜欢盲打，可以将其调至 0 (完全透明)；如果需要辅助对位，可以适当调高。
